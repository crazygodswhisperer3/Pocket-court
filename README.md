# 🏛 Pocket Court

Pocket Court is an offline-first legal navigator application built with a Rust-based core engine and a modular architecture. The goal of this project is to provide structured legal information in a clean, organized, and extensible way.

The core engine is implemented in Rust to ensure high performance, memory safety, reliability, and efficient resource usage. The project is designed with separation of concerns, keeping the legal logic independent from the user interface.

## 🦀 Why Rust?

The core system uses Rust because it provides:

- Memory safety without a garbage collector
- High performance and low resource consumption
- Strong type system for reliability
- Safe concurrency
- Minimal runtime overhead

Rust is ideal for building secure and efficient core logic systems.

## 📱 Building the Android APK

To generate an Android application (APK):

1. Clone this repository.
2. Open the project in Android Studio.
3. Ensure the Rust core is properly integrated.
4. Use the Android build system to compile the project.
5. Build the APK via:
   Build → Build Bundle(s) / APK(s) → Build APK.

Developers are encouraged to contribute an Android UI layer and integrate the Rust core to produce a fully functional APK.
